# Student-Performance-Analysis
# Objective
Identify factors influencing student academic performance.

# Key Findings
- Test preparation improves student scores.
- Higher parental education is associated with better performance.
- Standard lunch students generally perform better.
- Reading and writing scores show strong correlation.
- Demographic factors influence achievement patterns.
